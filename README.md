# Functions

## Normal Function Declaration

```javascript
function name(parameter) {
  // statement
}
```

## Function stored in a variable

```javascript
let name = function (parameter) {
  // statement
};
```

## Arrow Function

```javascript
const name = (parameter) => {
  // statement
};
```
